# Realtime Chat Application

A modern full-stack realtime chat application inspired by WhatsApp and Discord.

Built using the MERN stack with Socket.IO for realtime communication.

---

# Features

## Authentication
- Login & Signup
- JWT Authentication
- Protected Routes

## Realtime Messaging
- Instant messaging
- Online / Offline status
- Typing indicators
- Message reactions
- Edit & delete messages

## Voice & Media
- Voice messages
- Image sharing
- Video sharing
- File uploads

## Calling
- Audio calling
- Video calling
- Call history

## Group Features
- Group chats
- Add/remove members
- Group updates

## Notifications
- Browser notifications
- Realtime notification system

## UI Features
- Dark modern UI
- Theme support
- Responsive design
- WhatsApp-style chat experience

---

# Tech Stack

## Frontend
- React
- Vite
- Tailwind CSS
- Zustand
- Socket.IO Client

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Socket.IO
- Multer
